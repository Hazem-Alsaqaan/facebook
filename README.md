# facebook
Social App : look like facebook .. using Reactjs, mongodb, nodejs, express, redux-toolkit, react-router-dom
#### step 1. Clone repository in your server folder
` git clone https://github.com/Hazem-Alsaqaan/facebook.git `
#### step 2. Access server branch
` git checkout server `
#### step 3. Install dependencies
` npm install `
#### step 4. Run the server
` node server.js `
#### step 5. Go to client branch of this repository and follow steps in the README file
